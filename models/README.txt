This directory contains seismic model files in the following formats:

(1) ascii .tvel: depth, Vp, Vs, rho.

(2) binary .hed and .tbl: model tables for the usage of iaspei-tau/ttimes.

(3) binary .taup: model tables for the usage of TauP.
	To use addtional models in TauP, put the following line in a file in your home directory: ~/.taup
	taup.model.path=/opt/local/seismo/data/models
